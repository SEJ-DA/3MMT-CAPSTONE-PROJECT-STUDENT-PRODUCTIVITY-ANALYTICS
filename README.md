# Student Productivity & Distraction Analytics Dashboard

## 📊 Project Overview
This project analyzes how digital distractions impact student academic performance using data analytics and Power BI. It transforms raw student data into actionable insights that help educators identify at-risk students and improve learning outcomes.

## 🚨 Problem
Over 90% of students are highly distracted, mainly due to phone usage and social media.
Schools lack data-driven tools to measure productivity and identify disengaged students early.

## 💡 Solution
An interactive Power BI dashboard that:
Tracks student productivity scores
Analyzes distraction levels
Monitors study engagement
Identifies performance gaps


## ⚙️ Tools Used
- Power BI
- Excel
- Power QUERY
 
## 📈 Key Insight
📉 90.4% of students are highly distracted
📊 Focused students perform 34% better
⏱ Each additional study hour increases productivity by 7–8 points
